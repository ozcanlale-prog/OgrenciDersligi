# OgrenciDersligi


# 🎓 Fakültem | Öğrenci Yönetim Sistemi Arayüzü

Merhaba! Bu proje, üniversite öğrencileri için tasarladığım modern bir yönetim sistemi arayüzü çalışmasıdır. Bootstrap 5 kullanarak hem göze hitap eden hem de kullanışlı bir panel oluşturmayı hedefledim.

## ✨ Özellikler

*   **Giriş Paneli:** Şık bir giriş ekranı ve şifre sıfırlama alanı.
*   **Öğrenci Paneli:** Akademik ortalama, devamsızlık ve danışman bilgilerinin yer aldığı özet kartlar.
*   **Ders Takibi:** Dönemlik derslerin durumunu gösteren interaktif tablo.
*   **Dinamik Geçişler:** Sayfa yenilenmeden giriş ekranından ana panele geçiş sağlayan basit JavaScript mantığı.
*   **Karanlık Tema:** Göz yormayan, modern "Dark Mode" tasarımı.

## 🛠️ Neler Kullandım?

Bu projeyi geliştirirken şu teknolojilerden yararlandım:
*   **Bootstrap 5:** Tasarımın hızlı ve mobil uyumlu olması için.
*   **Google Fonts (Inter):** Daha okunaklı bir yazı tipi için.
*   **Bootstrap Icons:** Görselliği destekleyen simgeler için.
*   **JavaScript:** Ekranlar arası geçiş ve form simülasyonu için.

## 🚀 Nasıl Çalıştırılır?

Projeyi bilgisayarınıza indirip `index.html` dosyasını tarayıcınızda (Chrome, Edge vb.) açmanız yeterlidir. Herhangi bir kurulum gerektirmez.
